# Bookkeeping
# 簡易記帳程式-養成記帳好習慣
### 說明:一款使用Android Studio開發的記帳程式，這是我利用暑假看影片自學的作品，非課堂專題作業。使用者除了可以做簡單的支出與收入紀錄外，還能設置預算餘額、新增與查詢歷史紀錄、查看帳單詳情資訊、搜尋關鍵字查尋紀錄等功能。
